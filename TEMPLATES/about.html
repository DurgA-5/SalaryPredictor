ℹ️ About the Salary Predictor

An overview of the project's mission, technology, and developer.

Project Mission

The goal of the Employee Salary Prediction System is to harness the power of machine learning to provide transparent, data-driven salary insights. We aim to empower HR professionals, job seekers, and data analysts with a reliable tool to understand compensation trends in the Indian job market, fostering fairer and more informed salary negotiations.

System Architecture

  • Frontend: Built with Flask & Jinja2 for a server-rendered UI.
  • Backend Model: Trained using scikit-learn (Random Forest, XGBoost).
  • PDF Generation: Handled on the backend with `fpdf2` for robust reports.
  • Hosting: Deployed on cloud platforms like Render or PythonAnywhere.
  • User Interaction Flow:
    1. User enters employee data into the web form.
    2. Flask backend processes and validates input.
    3. ML model predicts salary based on pre-trained artifacts.
    4. Result is displayed on the page.
    5. User can download a personalized PDF report.

Machine Learning Pipeline

  • Data Cleaning: Handle missing values, remove outliers using IQR method.
  • Feature Engineering: One-hot encode categorical features, scale numeric data.
  • Feature Selection: Analyze correlation matrix and model-based importance.
  • Model Training: Experiment with Linear Regression, Random Forest, and XGBoost.
  • Evaluation: Validate model using R², Root Mean Squared Error (RMSE), and MAE.
  • Interpretability (Planned): Integrate SHAP values for prediction explanations.

Future Enhancements

  • API for Business Integration: A REST API for businesses to integrate predictions.
  • Resume Parsing: Use NLP to automatically extract features from uploaded resumes.
  • Time-Series Forecasting: Model salary growth trends over time.
  • Deeper Analytics: Interactive dashboards for market insights.
